    栈顶指针指向栈顶元素，自身占4个字节。

    空栈时将指针指向data[0]是不妥的，因为data[0]没有元素，所以将栈顶指针改为-1。

    基于理解的基础上自己分析采用何种写法。

    另一种方式：

    如果指针从top-1开始，先加后进，如果指针从top0开始，先进后加。

    共享栈：提升空间利用率。

    if(Q.rear==Q.front)
        return false;

    Q.front=(Q.front+1)%MaxSize;
    return true;
}

bool GetHead(SqQueue Q,ElemType &x){
    if(Q.rear==Q.front)
        return false;
    x=Q.data[Q.front];
    return true;
